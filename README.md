# Table Reservation System

A premium, pixel-perfect React-based application designed for high-end restaurants to manage table bookings, staff operations, and initial establishment setup. This system offers a seamless experience for both customers and restaurant staff, featuring a luxury-inspired aesthetic and highly interactive components.

## Overview

This project provides a comprehensive suite of tools for modern restaurant management:

- **Customer Facing:** An intuitive 4-step reservation wizard.
- **Staff Operations:** Real-time dashboards for monitoring table occupancy and guest arrivals.
- **Administration:** A dedicated setup flow for configuring the restaurant's physical layout and operational rules.

## Main Features

### 1. Customer Reservation Wizard

A streamlined, high-conversion flow for guests:

- **Date & Time Selection:** Integrated calendar with real-time slot availability.
- **Interactive Floor Map:** Visual table selection allowing guests to choose their preferred seating area (Window, Main Dining, Outdoor).
- **Guest Management:** Easy-to-use counter for party size.
- **Review & Payment:** Clear order summaries with integrated payment method selection.

### 2. Staff Management Dashboard

A powerful command center for daily operations:

- **Table Status:** Real-time visualization of available, occupied, and arriving guest tables.
- **Calendar View:** A detailed 24-hour timeline grid for managing bookings across different dining areas.
- **Day View:** A quick-glance list of all scheduled reservations with status updates.
- **Visit History:** Comprehensive logs of past guest dining experiences.

### 3. Setup & Onboarding Wizard

A 4-step guide to get the restaurant up and running:

- **Restaurant Details:** Configuration of address and operating hours.
- **Floor Map Configuration:** Easy CSV upload for table layouts, capacities, and types.
- **Operational Rules:** Toggle settings for table merging and walk-in policies.
- **Team Management:** Role-based invitations for staff members.

## Design & Aesthetics

The application features a **Premium Light Mode** integration (based on latest design specifications) while maintaining core support for luxury dark aesthetics.

- **Typography:** Modern, legible font stacks (Inter/Outfit).
- **Color Palette:** Professional neutral tones, gold accents for primary actions, and clear status color-coding.
- **Responsive Layouts:** Optimized for both handheld devices (POS terminals) and desktop management.

## Technology Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & Inline CSS Modules
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router 6](https://reactrouter.com/)
